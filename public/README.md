# Public Assets

This directory contains public assets for your website.

## CV/Resume
Place your resume PDF file as `cv.pdf` in this directory to display it on the Resume page.

**Current status:** The resume page is set up but requires you to add your CV PDF file.

**To add your CV:**
1. Save your resume as `cv.pdf`
2. Place it in this `/public/` directory
3. The resume page will automatically display your PDF

The resume page includes:
- PDF viewer for inline viewing
- Download button for easy access
- Mobile-responsive design
- Fallback for browsers that don't support PDF viewing
